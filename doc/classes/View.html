View Class

Defined in: source/js/view.js:1
Module: Mootor

The View class handles each screen of the application. A list of views is specified in the applications options and the files are loaded from the views/ folder. Each view has a viewName.js and a viewName.html file. The viewName.js file defines options for the view.
